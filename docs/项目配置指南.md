# 项目配置指南

本文档说明如何完成项目配置，包括API密钥获取和配置文件设置。

---

## 📋 配置步骤

### 步骤1：获取Gate.io API密钥

1. **登录Gate.io**
   - 访问：https://www.gate.io/
   - 使用你的账户登录

2. **进入API管理**
   - 点击右上角头像
   - 选择"账户管理" → "API密钥管理"
   - 或直接访问：https://www.gate.io/myaccount/apikeys

3. **创建新的API密钥**
   - 点击"创建API密钥"按钮
   - 填写标签（如：`crypto-monitor-system`）
   - 选择权限：
     - ✅ `Spot Trading`（现货交易）- 读取
     - ✅ `Futures Trading`（合约交易）- 读取
     - ✅ `Wallet` - 读取
     - ❌ 不要勾选"提现"权限（安全风险）
   - 设置IP白名单（可选，增强安全性）
   - 点击"确认"

4. **保存API密钥**
   - **API Key**：复制保存（只显示一次）
   - **API Secret**：复制保存（只显示一次）
   - ⚠️ 这些密钥只显示一次，务必保存到安全位置！

---

### 步骤2：获取企微Webhook URL

1. **登录企微管理后台**
   - 访问：https://work.weixin.qq.com/
   - 使用管理员账号登录

2. **进入群机器人配置**
   - 点击"应用管理"
   - 找到"群机器人"或"自定义机器人"
   - 或直接在具体群聊中添加机器人

3. **创建/获取Webhook URL**
   - 点击"添加机器人"
   - 选择要推送的群
   - 复制Webhook URL（格式如下）：
     ```
     https://qyapi.weixin.qq.com/cgi-bin/webhook/send?key=xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx
     ```

4. **保存Webhook URL**
   - 复制保存Webhook URL

## 🔧 测试配置

### 测试Gate.io API连接

创建测试脚本 `test_gateio_api.py`：

```python
#!/usr/bin/env python3
import requests
import time
import hmac
import hashlib
from datetime import datetime

API_KEY = "你的API_Key"
API_SECRET = "你的API_Secret"

def generate_signature(secret, message):
    """生成签名"""
    return hmac.new(
        secret.encode('utf-8'),
        message.encode('utf-8'),
        hashlib.sha512
    ).hexdigest()

def test_connection():
    """测试API连接"""
    # 测试公开端点（不需要签名）
    url = "https://api.gateio.ws/api/v4/spot/tickers"
    params = {'currency_pair': 'BTC_USDT'}
    
    try:
        resp = requests.get(url, params=params)
        if resp.status_code == 200:
            data = resp.json()
            print(f"✅ Gate.io API连接成功！")
            print(f"BTC/USDT 最新价格: ${float(data[0]['last']):,.2f}")
            return True
        else:
            print(f"❌ Gate.io API连接失败: {resp.status_code}")
            print(resp.text)
            return False
    except Exception as e:
        print(f"❌ 连接异常: {e}")
        return False

if __name__ == '__main__':
    test_connection()
```

运行测试：
```bash
cd D:/工作文件
python test_gateio_api.py
```

### 测试企微Webhook

创建测试脚本 `test_wecom_webhook.py`：

```python
#!/usr/bin/env python3
import requests
import json

WEBHOOK_URL = "你的Webhook_URL"

def test_webhook():
    """测试企微Webhook"""
    msg = {
        "msgtype": "text",
        "text": {

            print(f"✅ 企微Webhook推送成功！")
            print(f"请检查企微群是否收到消息")

            return False

        print(f"❌ 推送异常: {e}")
        return False

if __name__ == '__main__':
    test_webhook()
```

## ⚠️ 安全注意事项

1. **永远不要提交API密钥到仓库**
   - ✅ 已添加到 `.gitignore`
   - ✅ 使用环境变量或配置文件（不提交）

2. **定期更换API密钥**
   - 每30天更换一次
   - 发现泄露立即撤销

3. **使用最小权限原则**
   - API密钥只授予必要的权限
   - 不要授予"提现"权限

4. **启用IP白名单**
   - 在Gate.io API设置中启用IP白名单
   - 只允许特定IP访问API

## 📞 故障排查

### Gate.io API连接失败

**问题**：`requests.exceptions.SSLError` 或 `Connection timeout`

**解决方案**：
1. 检查网络连接
2. 使用代理（如需要）
3. 检查API端点URL是否正确

### 企微Webhook推送失败

**问题**：返回 `errcode: 93000`（webhook配置错误）

**解决方案**：
1. 检查Webhook URL是否正确
2. 确认机器人未被移除
3. 重新创建机器人获取新URL
